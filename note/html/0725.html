스프레드연산자(깊은 복사,얕은 복사)

< Spread 연산자 >
두 개의 배열을 하나의 배열로 병합하려고 할 때 사용할 수 있음
...뒤에 오는 array를 펼쳐주고, [] 안에 펼쳐진 값을 포함하여 여러값을 넣으면 하나로 합칠 수 있음


< 깊은 복사, 얕은 복사 >
얕은 복사는 객체의 주소 값을 복사하고, 깊은 복사는 객체의 실제 값을 복사함
Spread 연산자는 얕은 복사에 해당됨


< useState, useEffect, useRef, useReducer >

useState: 컴포넌트에 state 변수를 추가할 수 있게함
초기값을 설정할 수 있고, 이를 변경할 수 있음

useEffect: 무언가가 변경되었을 때 실행됨
의존성 배열이 존재하지 않는다면 라이프사이클에서 마운트를 제외한 모든 경우,
의존성 배열의 []이 비워져 있다면 마운트인 경우,
의존성 배열의 []에 값이 있다면 해당되는 값이 변경되는 경우 등에서 실행됨

useRef: 렌더링에 필요하지 않은 값을 참조할 수 있음
전역변수와 비슷하여 값을 가지게 할 수 있음

useReducer: 여러 컴포넌트를 switch문과 함께 적절히 구성하여 사용할 수 있음
state, action을 지정하고, 이를 이용하여 같은 값을 하눈에 보기 쉽도록 함
각 컴포넌트에 type을 지정하고, action을 만들 수 있음
